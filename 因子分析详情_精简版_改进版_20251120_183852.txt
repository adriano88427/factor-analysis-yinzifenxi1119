1. 评分标准
==================================================

本报告采用多维度综合评分体系，总分10分，包含四个核心评价指标：

1.1 评分指标及权重分布
==================================================

• 多空收益评分：35%权重 (权重最高，衡量收益能力)
• IC均值评分：30%权重 (衡量预测能力)
• IR值评分：25%权重 (衡量稳定性)
• 统计显著性评分：10%权重 (衡量可靠性)

评分公式：综合得分 = 多空收益得分(0-4分)×0.35 + IC均值得分(0-4分)×0.30 + IR值得分(0-2.5分)×0.25 + 统计显著性得分(0-1分)×0.10

1.2 具体评分标准详解
==================================================

【多空收益评分标准】(0-4分，权重35%)
----------------------------------------
该指标衡量因子带来的实际收益能力，数值越高表明因子对投资的贡献越大：

• 4.0分：多空收益≥4.0% (卓越表现，强烈推荐)
• 3.5分：多空收益≥4.0% + 超额奖励机制 (每超1%额外0.5分，最高4.0分)
• 3.0分：3.0% ≤ 多空收益 < 4.0% (良好表现)
• 2.5分：2.0% ≤ 多空收益 < 3.0% (中等表现)
• 1.5分：1.0% ≤ 多空收益 < 2.0% (较弱表现)
• 1.0分：0.0% ≤ 多空收益 < 1.0% (微弱表现)
• 0.5分：多空收益 < 0.0% (负向表现)

【IC均值评分标准】(0-4分，权重30%)
----------------------------------------
该指标衡量因子的预测能力，IC(Information Coefficient)反映因子值与收益的相关性：

• 4.0分：|IC均值|≥0.12 (优秀预测能力)
• 3.5分：|IC均值|≥0.12 + 超额奖励机制 (每超0.03额外0.5分，最高4.0分)
• 3.0分：0.10 ≤ |IC均值| < 0.12 (强预测能力)
• 2.0分：0.05 ≤ |IC均值| < 0.10 (中等预测能力，有效阈值)
• 1.0分：0.02 ≤ |IC均值| < 0.05 (弱预测能力)
• 0.5分：|IC均值| < 0.02 (极弱预测能力)

【IR值评分标准】(0-2.5分，权重25%)
----------------------------------------
该指标衡量因子预测能力的稳定性，IR(Information Ratio)=IC均值/IC标准差：

• 2.5分：|IR值|≥1.5 (极强稳定性)
• 2.0分：1.0 ≤ |IR值| < 1.5 (强稳定性)
• 1.5分：0.5 ≤ |IR值| < 1.0 (中等稳定性)
• 1.0分：0.3 ≤ |IR值| < 0.5 (弱稳定性)
• 0.8分：0.15 ≤ |IR值| < 0.3 (较弱稳定性)
• 0.5分：|IR值| < 0.15 (极弱稳定性)

【统计显著性评分标准】(0-1分，权重10%)
----------------------------------------
该指标衡量结果的统计学可信度，避免偶然性结果：

• 1.0分：p值 < 0.01 (高度显著，99%置信度)
• 0.8分：0.01 ≤ p值 < 0.05 (显著，95%置信度)
• 0.6分：0.05 ≤ p值 < 0.1 (边缘显著，90%置信度)
• 0.3分：p值 ≥ 0.1 (不显著，结果可能存在偶然性)

1.3 评级体系映射
==================================================

综合得分转换为字母评级，便于理解和使用：

• A+级：综合得分≥4.0分 (卓越级别，强烈推荐使用)
• A级：3.0分≤综合得分<4.0分 (优秀级别，推荐使用)
• B+级：2.0分≤综合得分<3.0分 (良好级别，可考虑使用)
• B级：1.5分≤综合得分<2.0分 (一般级别，谨慎使用)
• C+级：1.0分≤综合得分<1.5分 (较弱级别，不推荐使用)
• C级：0.5分≤综合得分<1.0分 (弱级别，不建议使用)
• D级：综合得分<0.5分 (无效级别，避免使用)

1.4 特殊评级机制
==================================================

针对表现优异的因子，系统采用特殊评级机制：

• 前10日最大涨幅类优秀因子：如果IC均值>0.12且多空收益>0.04，给予A-或A评级
• 因子方向识别：自动识别正向因子(做多)和负向因子(反向使用)
• 风险等级评估：根据综合得分自动判断风险等级(低/中/高)

1.5 投资建议等级
==================================================

根据评级给出相应的投资建议：

• A+级：强烈推荐使用，权重15-25%
• A级：推荐使用，权重8-15%
