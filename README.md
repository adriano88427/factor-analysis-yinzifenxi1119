# 因子分析项目

这是一个用于进行因子分析的Python项目，包含数据处理、统计分析和可视化功能。

## 功能特点

- 因子数据加载与预处理
- 多种因子分析方法
- 统计显著性检验
- 结果可视化
- 日志记录功能

## 主要文件

- `yinzifenxi1119.py` - 主要的因子分析代码
- `因子分析日志_*.txt` - 分析过程日志文件
- `因子分析汇总_*.csv` - 分析结果汇总数据
- `因子分析详情_精简版_*.txt` - 精简版分析报告

## 使用方法

1. 确保安装了所需的Python包：
   ```
   pip install pandas numpy scipy matplotlib seaborn
   ```

2. 运行主程序：
   ```
   python yinzifenxi1119.py
   ```

3. 查看生成的日志文件和结果文件

## 数据文件

默认使用的数据文件是 `创业板单日下跌14%详细交易日数据（清理后）1114.xlsx`，确保该文件在项目目录中。

## 环境要求

- Python 3.7+
- pandas
- numpy
- scipy (可选，用于高级统计功能)
- matplotlib 和 seaborn (可选，用于可视化)

## 注意事项

- 程序会自动生成日志文件，记录分析过程
- 如果缺少某些依赖包，程序会给出警告但会继续运行
- 大部分功能都有错误处理，确保程序稳定性
